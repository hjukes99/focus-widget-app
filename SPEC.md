## SPEC.md

**App Name:** Focus Widget

**Target User:** Individuals seeking to improve focus and manage time effectively.

**MVP:** A minimal desktop widget that displays a countdown timer and allows users to set focus intervals (e.g., 25 minutes) followed by short breaks (e.g., 5 minutes). It should provide basic visual feedback for the timer state (running, break, finished).

**Core Features:**
- Configurable focus interval duration.
- Configurable break interval duration.
- Visual countdown timer.
- Start/Pause/Reset controls.
- Optional sound notification upon completion.

**Future Enhancements (Post-MVP):**
- Customizable themes/appearances.
- Session history logging.
- Integration with task management tools.
- Pomodoro technique support (longer breaks after multiple focus sessions).
